# Mai Abd El-hameed - Frontend Developer Portfolio

A modern, responsive portfolio website showcasing my skills, projects, and experience as a Frontend Developer. Built with vanilla HTML, CSS, and JavaScript, featuring smooth animations, interactive elements, and a fully functional contact form.

## Features

- **Animated Hero Section**: Eye-catching landing page with typing animations and gradient effects

- **Project Portfolio**: Detailed showcase of featured projects with live demos and GitHub links
- **Working Contact Form**: Integrated with Web3Forms API for direct email communication

- **Fully Responsive**: Mobile-first design that works seamlessly across all devices

## Setup & Installation

1. Clone the repository:
```bash
git clone https://github.com/MaiAbdulhamid/portfolio.git
```

2. Navigate to the project directory:
```bash
cd portfolio
```

3. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js http-server
npx http-server
```

4. Visit `http://localhost:8000` in your browser

## Contact Form Setup

The contact form uses Web3Forms API. To set it up:

1. Visit [Web3Forms](https://web3forms.com)
2. Sign up with your email
3. Get your access key
4. Replace the access key in `index.html` (line 404)

The form is already configured with my email: mai.abdelhameed95@gmail.com

## Key Features Explained

### Smooth Scrolling
All navigation links feature smooth scrolling to sections with `scroll-behavior` implementation.

Fade-in animations triggered by scroll position using the Intersection Observer API.

### Contact Form
- Real-time form validation
- Loading, success, and error states
- Direct email delivery via Web3Forms API
- No backend required
